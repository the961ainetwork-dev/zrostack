# ZHIVE — AI Business Suite for MENA Entrepreneurs

## Deploy
1. Push to GitHub
2. Import to vercel.com → Deploy
3. Add ANTHROPIC_API_KEY environment variable in Vercel settings
4. Run setup.sql in Supabase

## Pages
- / — Landing page (home, features, pricing, about, blog, FAQ, GDPR, legal)
- /bizoptimizer.html — Business Optimizer AI (5 agents)
- /api/claude — Serverless AI proxy
